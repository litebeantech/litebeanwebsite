Tax / Discount / Other Charges Configurations

Navigation

On the Left menu click, Inventory >> Configure Unit Tax

Second table (first table on right) - is for configuring Tax / Discount / Other Charges



Click - to add new row
Click - to edit

Types - Types of charges
  • Tax
  • Discount
  • Charges
  • Charges
Value - Value (numeric) along with Value Type (column) define the charge amout

Value Type - Defines how the value (column) will be used
  • Percent
  • Flat
Operation - Should added/substracted to/from Applied On (column)
  • ADD
  • SUB
Applied On - Where this tax/discount/charge should be applied - default it will be total, but it can be applied on any other charge we created
